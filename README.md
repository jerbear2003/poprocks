# poprocks

I want a website that ranks the different Pop Tarts flavors and puts them up against Pop Rocks flavors. Pop Rocks should always lose the competition to Pop Tarts. I want a really fun website with an almost psychedelic sticker look. I want a page that just at random provides a different picture of a Pop Tart and a page for in-depth reviews of a Pop Tart flavor against a Pop Rocks flavor. It would be fun if at least every time the home page is reloaded there’s a different color scheme or theme going on. It should feel very grungy and DIY
